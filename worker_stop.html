Stop a worker

Usage

worker_stop(queue, worker_id, type = "message", host = "127.0.0.1", port = 6379)

Arguments

queue
Name of the queue
worker_id
Name of a single worker
type
Either "message" to send a message or "kill" or kill the worker.
host
Redis hostname
port
Redis port

Description

Try to stop a worker. You won't need to call this very often, hopefully.
